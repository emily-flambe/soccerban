# Soccerban

A Sokoban-style puzzle game with a soccer twist. Push all the balls into the goals to win each level.

## Play

**Live**: [soccerban.emilycogsdill.com](https://soccerban.emilycogsdill.com)

**Controls**: Arrow keys or WASD to move, R to restart the level.

## Development

```bash
npm run dev      # local dev server (wrangler)
npm run deploy   # deploy to Cloudflare Workers
```

## Stack

- Vanilla HTML/JS/CSS — no build step, no frameworks
- Canvas-based rendering
- Deployed to Cloudflare Workers (static assets)
